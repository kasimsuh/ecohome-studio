# Budget Tradeoffs

- Start with passive design, envelope quality, and layout efficiency before layering on expensive technology.
- If the budget is tight, use phased upgrade planning so solar, storage, or advanced systems can be added later.
- Reserve roof, panel, and utility space for future clean-energy upgrades even when they are not installed on day one.
- Keep the plan compact and the plumbing or mechanical cores organized so construction complexity stays manageable.
